### Features

1. Can minify JSON
   - ctrl/cmd + p
   - Search for **Minify JSON**
   - Press `Enter`
2. Can Beautify (prettify) JSON
   - ctrl/cmd + p
   - Search for **Prettify JSON**
   - Press `Enter`
