### Features

1. Can minify JSON
   a. ctrl/cmd + p
   b. Search for **Minify JSON**
   c. Press `Enter`
2. Can Beautify (prettify) JSON
   a. ctrl/cmd + p
   b. Search for **Prettify JSON**
   c. Press `Enter`
